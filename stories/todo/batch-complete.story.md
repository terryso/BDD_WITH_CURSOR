# 批量完成待办事项

## 作为（As a）
命令行用户

## 我想要（I want）
能够一次性将多个待办事项标记为已完成

## 以便于（So that）
提高工作效率，避免重复执行完成命令

## 验收标准（Acceptance Criteria）
- [ ] 支持通过命令行一次性完成多个待办事项
- [ ] 命令格式正确且参数验证完整
- [ ] 提供完整的成功反馈信息
- [ ] 正确处理无效的待办事项ID
- [ ] 保持未指定ID的待办事项状态不变
- [ ] 正确处理已完成的待办事项

## 补充信息
- 业务规则：
  * 支持同时完成多个待办事项
  * 忽略已完成的待办事项
  * 忽略不存在的待办事项ID
  * 至少需要提供一个有效的待办事项ID
  * 所有操作在一个事务中完成，要么全部成功，要么全部失败

- 技术要求：
  * 命令格式：`node dist/cli.js complete-batch <id1> <id2> ... <idN>`
  * 参数验证：所有ID必须为正整数
  * 错误处理：提供清晰的错误提示
  * 成功反馈：显示所有成功完成的待办事项信息
  * 数据一致性：确保批量操作的原子性
